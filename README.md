# 🐍 Python Collections Project

A clean and beginner-friendly Python project that demonstrates the use of core collection data types in Python. This project helps understand how data can be stored, managed, and manipulated efficiently.

---

## 🚀 Features

- 📋 List – Ordered, mutable collection  
- 🔗 Set – Unordered, unique elements  
- 📦 Tuple – Ordered, immutable collection  
- 🗂️ Dictionary – Key-value data structure
